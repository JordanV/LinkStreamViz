LinkStreamViz
==========

Simple tool to draw link streams into SVG files.

Before using
------------

Make sure the svgfig module is installed. If not:
```bash
cd-to-LinkStreamViz-repo
cd svgfig/
python setup.py install --user
```

Usage
----------
```python
python main.py <input_file>
```
